## ☁️ Deployment Guide

### 🚀 Production Deployment
1. **GitHub Sync:** Push your `evaluations.db` and code to a repository.
2. **Streamlit Cloud:** Connect your repo at [share.streamlit.io](https://share.streamlit.io).
3. **One-Click Launch:** Deploy `app.py`.

### 🧪 Local Setup
```bash
# Clone
git clone <repository_url>

# Setup Engine
pip install -r requirements.txt

# Execute
streamlit run app.py
```
